ECHO ===== MUTATION INDEX COHERENCE CANARY V1 START =====

DO SANDBOX
USE STUDENTS
SET INDEX TO STUDENTS
SET ORDER TO TAG LNAME

ECHO ===== BASELINE =====
COUNT
RECNO
SMARTLIST 5

ECHO ===== APPEND TEST RECORD =====
APPEND
REPLACE SID WITH 59999999
REPLACE LNAME WITH "ZTEST"
REPLACE FNAME WITH "INDEX"
REPLACE GENDER WITH "X"
REPLACE MAJOR WITH "CSCI"
REPLACE GPA WITH 3.50
REPLACE EMAIL WITH "ztest.index@student.mcc.edu"

ECHO ===== VERIFY APPENDED RECORD =====
RECNO
TUPLE *

ECHO ===== SEEK NEW KEY =====
SEEK ZTEST
RECNO
TUPLE *

ECHO ===== FIND NEW KEY =====
FIND ZTEST
RECNO
TUPLE *

ECHO ===== REPLACE INDEXED KEY ZTEST TO ATEST =====
REPLACE LNAME WITH "ATEST"
RECNO
TUPLE *

ECHO ===== SEEK OLD KEY SHOULD FAIL =====
SEEK ZTEST
RECNO
TUPLE *

ECHO ===== SEEK NEW KEY SHOULD FIND =====
SEEK ATEST
RECNO
TUPLE *

ECHO ===== DELETE TEST RECORD =====
DELETE
RECNO
TUPLE *

ECHO ===== SEEK DELETED KEY =====
SEEK ATEST
RECNO
TUPLE *

ECHO ===== RECALL TEST RECORD =====
RECALL
RECNO
TUPLE *

ECHO ===== SEEK RECALLED KEY =====
SEEK ATEST
RECNO
TUPLE *

ECHO ===== PREDICATE CHECKS =====
COUNT FOR LNAME = "ATEST"

SCAN FOR LNAME = "ATEST"
    RECNO
ENDSCAN

SMARTLIST FOR LNAME = "ATEST"

DELETE
