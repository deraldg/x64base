Welcome to x64base

Overview

x64base is the public home for DotTalk++, related xBase research, educational material, technical documentation, and long-term project direction. This first skeleton is meant to be small, credible, and easy to maintain under IIS with ordinary static files.

Canonical Source

Public source repository: github.com/deraldg/x64base

Current Intent

Current Build a restrained technical site that supports three public jobs at once: explain the project, document the software, and preserve the larger xBase tradition that DotTalk++ is carrying forward.
