## Overview

Arctic is the current code name for the DotTalk++ Workbench TUI surface.

This page is kept for older links. The current product lane is
[Parallel GUI/TUI](/products/parallel-gui-tui), and the current documentation
page is [Arctic TUI](/docs/talk-family/arctic).

The family group is **DotTalk++ Workbench**. Arctic should not be used as the
umbrella name for every GUI, browser, shell, and workflow surface.

In the local source tree, the real surfaces include:

- Arctic TUI and shell-bridge work where the TUI is the active surface.
- TUI tuple grids, record popups, command panes, and menu-driven runtime inspection.
- Workbench evidence where the visible runtime identifies the Arctic/TUI code-name surface.

Use DotTalk++ Workbench for the broader family. Use Arctic for the TUI code
name.
