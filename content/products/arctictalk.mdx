## Overview

ArcticTalk is the code name for the DotTalk++ Workbench TUI surface.

The family group is **DotTalk++ Workbench**. ArcticTalk should not be used as
the umbrella name for every GUI, browser, shell, and workflow surface.

Its subject is the terminal/TUI layer around the runtime, not a claim that every
Workbench surface is ArcticTalk.

In the local source tree, the real surfaces include:

- ArcticTalk/Foxtalk TUI and shell-bridge work where the TUI is the active surface.
- TUI tuple grids, record popups, command panes, and menu-driven runtime inspection.
- Workbench evidence where the visible runtime identifies the Arctic/TUI code-name surface.

Use DotTalk++ Workbench for the broader family. Use ArcticTalk for the TUI code
name.

See [ArcticTalk docs](/docs/talk-family/arctictalk),
[Current Project Truth](/docs/dev/project-truth), and
[SelfDoc Website Publication](/docs/dev/selfdoc-website-publication), plus the
canonical public repo [deraldg/x64base](https://github.com/deraldg/x64base).
