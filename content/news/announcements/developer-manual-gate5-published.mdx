The reviewed DotTalk++ / x64base developer manual has reached the website
integration stage from public source commit `be935053`.

The accepted snapshot contains:

- 4,118 manual lines and 237 headings;
- 24 sections and four appendices;
- 183 standalone command-reference pages;
- 164 reader-linked pages and 19 supplemental pages;
- 3,974 command lineage rows.

These counts describe the reviewed documentation snapshot. They are not a
claim that every runtime lane is complete or that documentation generation is
fully automatic.
