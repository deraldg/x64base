x64base uses a hybrid model that separates:

- **Core engine** (x64base Engine) — MIT-licensed
- **Learning language** (DotTalk++) — educational license
- **Free/collaboration campus** (Laboratory Campus / LabTalk) — non-profit license

This page is a plain-language overview. The authoritative terms live in the individual license pages.

## Quick guidance (non-legal)

- If you want the engine for general development, see the [MIT license](/licensing/mit-license-engine).
- If you want DotTalk++ for instruction, see the [educational license](/licensing/dottalk-educational-license).
- If you are a non-profit, see the [Laboratory Campus / LabTalk license](/licensing/labtalk-non-profit-license).
