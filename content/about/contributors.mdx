## Primary Author

**Derald R Grimwood Jr.** is the originator and primary author of the x64base, DotTalk++, and LabTalk concept line. The project preserves the xBase lineage while using modern C++ implementation work, self-documenting contracts, and an education-first laboratory model.

## Contributors

**[Nathaniel "Nate" Grimwood](https://www.linkedin.com/in/nathan-grimwood-36499286/), IT Director / Director of Managed Services** is recognized as a human contributor who provided AI-related advice and guidance to Derald R Grimwood Jr. during the LabTalk/x64base development work.

The contributor model for this project treats AI-assisted development as part of the engineering record. Human contributors may advise on AI use, source organization, implementation planning, documentation, website publication, and how source contracts connect to public-facing explanations. Human review remains the authority for release decisions, authorship, licensing, and trademark use.

## Contribution Record

LabTalk is being developed as a living laboratory campus. Contributions are expected to leave a trail through one or more of these surfaces:

- Source code and build scripts.
- `@dottalk.usage`, `@dottalk.contract`, and related SelfDoc contracts.
- Case stories and runtime proofs.
- Website pages generated from the vertical documentation system.
- Lab exercises, education features, and GUI/portal experiments.

This page is a recognition surface, not a substitute for source control history, license notices, or third-party acknowledgements.
