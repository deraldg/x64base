## Accuracy rule

The website reports progress; it does not promote implementation truth by
itself. A local or experimental lane stays labeled until its source, contracts,
tests, review state, and public commit are available. New public pages should
be inspected alongside neighboring project-truth, current-lanes, downloads,
manual, and metadata pages so a correct new statement does not leave stale
contradictions nearby.
