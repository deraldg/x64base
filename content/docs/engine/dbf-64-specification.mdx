## Table header (conceptual)

- magic/version
- record count
- header length
- record length
- flags / reserved

## Field descriptors

Each field descriptor defines:

- name
- type
- length
- decimals (if numeric)
- offsets / flags

## Records

- fixed-length records
- deleted flag
- field-aligned payload
