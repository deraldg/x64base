## Suggested onboarding

1. Start with [Getting Started](/docs/getting-started/overview)
2. Use [TableTalk](/docs/talk-family/tabletalk) to learn schema and records
3. Use [DotTalk++](/docs/dottalk/language-guide) for scripted workflows
4. Review the [LabTalk license](/licensing/labtalk-non-profit-license)

## Best practices

- keep sample datasets small and inspectable
- use scripts to make lessons repeatable
