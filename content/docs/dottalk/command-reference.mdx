## Authority and regeneration

This landing page is a reviewed derivative. Public source, runtime behavior,
HELP/CMDHELP, usage contracts, and the accepted manualgen artifacts remain the
technical evidence.

When command registration, HELP, contracts, or manual sections change, the
reference must be reharvested, validated, reviewed, and accepted before this
snapshot is advanced. Website prose alone cannot promote a command or change
its support status.
