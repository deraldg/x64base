## What it does

ArcticTalk is the code name for the DotTalk++ Workbench TUI surface.

The family group is **DotTalk++ Workbench**. ArcticTalk should not be used as
the umbrella name for all GUI, browser, shell, and workflow surfaces.

Its subject is the terminal/TUI presentation layer around the runtime:

- TUI menus and panels
- tuple grids and record popups
- command/output panes in the TUI shell
- keyboard-driven workbench interaction

Runtime screenshots for the current ArcticTalk TUI evidence lane are collected in the [LabTalk Runtime Evidence Gallery](/docs/labtalk/runtime-evidence).

Real local command surfaces and workflow anchors include:

```text
BUILDLMDB
BUILDLMDB CLEAN YES
BUILDLMDB FORCE YES
REINDEX
CMDHELPCHK
CMDHELPCHK REF
WORKSPACE OPEN DBF
WORKSPACE OPEN DBF CDX
WORKSPACE SAVE <name>
WORKSPACE LOAD <name>
```

DotScript scripts, smoke checks, maintenance lanes, GUI previews, and generated
proposal packages belong under the broader DotTalk++ Workbench family, not under
ArcticTalk unless they are specifically TUI-facing.

See [Current Project Truth](/docs/dev/project-truth) and
[SelfDoc Website Publication](/docs/dev/selfdoc-website-publication), plus the
canonical public repo [deraldg/x64base](https://github.com/deraldg/x64base).
