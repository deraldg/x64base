## What it does

Arctic is the code name for the DotTalk++ Workbench TUI surface.

The family group is **DotTalk++ Workbench**. Arctic should not be used as the
umbrella name for all GUI, browser, shell, and workflow surfaces.

Its subject is the terminal/TUI presentation layer around the runtime:

- TUI menus and panels.
- Tuple grids and record popups.
- Command/output panes in the TUI shell.
- Keyboard-driven workbench interaction.

Runtime screenshots for the current Arctic TUI evidence lane are collected in
the [LabTalk Runtime Evidence Gallery](/docs/labtalk/runtime-evidence).

Real local command surfaces and workflow anchors include:

```text
BUILDLMDB
BUILDLMDB CLEAN YES
BUILDLMDB FORCE YES
REINDEX
CMDHELPCHK
CMDHELPCHK REF
WORKSPACE OPEN DBF
WORKSPACE OPEN DBF CDX
WORKSPACE SAVE <name>
WORKSPACE LOAD <name>
```

DotScript scripts, smoke checks, maintenance lanes, GUI previews, and generated
proposal packages belong under the broader DotTalk++ Workbench family, not under
Arctic unless they are specifically TUI-facing.

See [Parallel GUI/TUI](/docs/talk-family/parallel-gui-tui),
[Current Project Truth](/docs/dev/project-truth), and
