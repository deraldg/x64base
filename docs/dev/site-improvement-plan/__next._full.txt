<h3 id="3-academic-and-education-review"><a class="no-underline" href="#3-academic-and-education-review">3. Academic and education review</a></h3>
<p>Status: opening soon.</p>
<p>The <a href="/docs/labtalk/academic-positioning">Laboratory Campus Academic Positioning and Research Agenda</a>
now records the proposed research framing, candidate contributions, research
questions, evidence needs, and claim boundaries for this lane.</p>
<p>The project is preparing to invite academic interest. Desired input includes:</p>
<ul>
<li>CS professors for algorithms, data structures, systems programming, runtime design, and language lessons.</li>
<li>database professors for relational theory, DBF lineage, indexing, locking, triggers, buffering, and mutation safety.</li>
<li>technical writing teachers for documentation standards, manual structure, comment collection, terminology, and student-facing explanations.</li>
<li>general education specialists for prerequisites, accessibility, assessment, and the shape of the Laboratory Campus.</li>
<li>instructors from different schools of learning to review whether lessons work through reading, repetition, diagrams, command practice, and project work.</li>
</ul>
<h3 id="4-documentation-proof-gates"><a class="no-underline" href="#4-documentation-proof-gates">4. Documentation proof gates</a></h3>
<p>Status: active policy.</p>
<p>Website updates should retain the same proof classes used by the documentation matrix:</p>
<ul>
<li>runtime-evidenced</li>
<li>source-evidenced</li>
<li>help-catalog-evidenced</li>
<li>generated-reviewed</li>
<li>planned-or-in-progress</li>
<li>review-needed</li>
</ul>
<p>A missing polished lesson is a deficiency. A planned lane labeled planned is not a deficiency unless the site implies it is finished.</p>
