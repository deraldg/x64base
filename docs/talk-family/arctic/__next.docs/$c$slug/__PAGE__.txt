8:T5c8,<h2 id="what-it-does"><a class="no-underline" href="#what-it-does">What it does</a></h2>
<p>Arctic is the code name for the DotTalk++ Workbench TUI surface.</p>
<p>The family group is <strong>DotTalk++ Workbench</strong>. Arctic should not be used as the
umbrella name for all GUI, browser, shell, and workflow surfaces.</p>
<p>Its subject is the terminal/TUI presentation layer around the runtime:</p>
<ul>
<li>TUI menus and panels.</li>
<li>Tuple grids and record popups.</li>
<li>Command/output panes in the TUI shell.</li>
<li>Keyboard-driven workbench interaction.</li>
</ul>
<p>Runtime screenshots for the current Arctic TUI evidence lane are collected in
the <a href="/docs/labtalk/runtime-evidence">LabTalk Runtime Evidence Gallery</a>.</p>
<p>Real local command surfaces and workflow anchors include:</p>
<pre><code class="language-text">BUILDLMDB
BUILDLMDB CLEAN YES
BUILDLMDB FORCE YES
REINDEX
CMDHELPCHK
CMDHELPCHK REF
WORKSPACE OPEN DBF
WORKSPACE OPEN DBF CDX
WORKSPACE SAVE &#x3C;name>
WORKSPACE LOAD &#x3C;name>
</code></pre>
<p>DotScript scripts, smoke checks, maintenance lanes, GUI previews, and generated
proposal packages belong under the broader DotTalk++ Workbench family, not under
Arctic unless they are specifically TUI-facing.</p>
