6:T7bb,<p>DotTalk++ is an interactive command shell over the x64base runtime. You type a
command, it runs immediately against the current work area, and the result — data
or a <code>cmd:</code> status line — comes straight back. The same commands run
non-interactively through the <a href="/docs/dottalk/dotscript-language-guide">DotScript</a>
runner.</p>
<p>A REPL makes the engine tangible: open a table, inspect its structure, move the
record pointer, filter, and aggregate — watching each step.</p>
<h2 id="a-typical-session"><a class="no-underline" href="#a-typical-session">A typical session</a></h2>
<pre><code class="language-text">DotTalk++> USE STUDENTS
DotTalk++> COUNT
1000000
DotTalk++> STRUCT
DotTalk++> GO TOP
DotTalk++> DISPLAY FNAME, LNAME, MAJOR, GPA
DotTalk++> SET FILTER TO MAJOR = "CSCI"
DotTalk++> COUNT
90700
DotTalk++> AGGS ALL GPA
COUNT=90700 SUM=... AVG=... MIN=2 MAX=4
DotTalk++> SET FILTER TO
DotTalk++> QUIT
</code></pre>
<h2 id="useful-exploration-commands"><a class="no-underline" href="#useful-exploration-commands">Useful exploration commands</a></h2>
<ul>
<li><code>USE</code>, <code>CLOSE</code>, <code>SELECT</code> — open, close, and switch work areas.</li>
<li><code>STRUCT</code>, <code>STATUS</code>, <code>COUNT</code> — inspect structure, area state, and row counts.</li>
<li><code>GO TOP</code> / <code>GO BOTTOM</code> / <code>SKIP</code>, <code>SEEK</code>, <code>LOCATE</code> — move the record pointer.</li>
<li><code>DISPLAY</code>, <code>LIST</code>, <code>SMARTLIST</code> — show records.</li>
<li><code>SET FILTER TO</code>, <code>SUM</code> / <code>AVG</code> / <code>AGGS ALL</code> — scope and aggregate.</li>
<li><code>HELP</code>, <code>CMDHELP</code>, <code>PREDHELP</code> — command, and predicate help.</li>
</ul>
