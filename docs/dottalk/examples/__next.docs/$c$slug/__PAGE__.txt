<h2 id="open-inspect-and-list"><a class="no-underline" href="#open-inspect-and-list">Open, inspect, and list</a></h2>
<pre><code class="language-text">USE STUDENTS
STRUCT
COUNT
LIST NEXT 5 FNAME, LNAME, MAJOR, GPA
</code></pre>
<h2 id="filter-and-aggregate-a-subset"><a class="no-underline" href="#filter-and-aggregate-a-subset">Filter and aggregate a subset</a></h2>
<pre><code class="language-text">USE STUDENTS
SET FILTER TO MAJOR = "CSCI"
COUNT
AGGS ALL GPA
SET FILTER TO
</code></pre>
<p><code>AGGS ALL</code> returns <code>COUNT / SUM / AVG / MIN / MAX</code> in a single scan. <code>SET FILTER TO</code> with no argument clears the filter.</p>
<h2 id="order-then-seek"><a class="no-underline" href="#order-then-seek">Order, then seek</a></h2>
<pre><code class="language-text">USE STUDENTS
CDX CREATE
INDEX ON LNAME TAG LNAME
BUILDLMDB
SET ORDER TO LNAME
SEEK "Quinn"
DISPLAY
</code></pre>
<p><code>SEEK</code> on the active CDX tag uses the LMDB keyed range-seek (O(log n)); ordered
navigation (<code>TOP</code>, <code>BOTTOM</code>, <code>SKIP</code>) reads the index cursor directly.</p>
<h2 id="copy-a-filtered-projection-to-a-new-table"><a class="no-underline" href="#copy-a-filtered-projection-to-a-new-table">Copy a filtered projection to a new table</a></h2>
<pre><code class="language-text">USE STUDENTS
SET FILTER TO GPA >= 3.5
COPY TO HONORS FIELDS FNAME, LNAME, MAJOR, GPA
SET FILTER TO
USE HONORS
COUNT
</code></pre>
