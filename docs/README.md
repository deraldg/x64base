# DotTalk++ / x64base Documentation Factory

This documentation estate is managed by the Master Document Organizer and coordinated with SelfDoc.

Doctrine:

```text
Runtime proves.
Source defines.
HELP explains.
Metadata organizes.
CMDHELPCHK validates.
SelfDoc preserves provenance.
Master Document Organizer assembles teachable manuals.
```

This tree separates governance, evidence, topic packets, manuals, glossary, review items, diagrams, screenshots, transcripts, scripts, archive material, and publish outputs.

The manuals do not decide what is true independently. They render the same verified truth differently.

Durable contracts are indexed under `contracts/`. If a rule is meant to
constrain future work, it should be registered there rather than preserved only
in chat, a temporary report, or an isolated planning note.
