# Blackbox Native Surface Addendum

Blackbox is the educational model for data entering a system, processing occurring inside the blackbox, and information coming out.

The current BBOX surface is native C++ so it behaves like the rest of DotTalk++ and can participate in HELP, CMDHELP, source usage contracts, and runtime smoke tests.

External Python helpers may support Blackbox lane mapping, but BBOX itself is a runtime command/application surface.
