a:T735,<p>The local project is command-shell first. Use DotTalk++ scripts and commands rather than invented package-style examples.</p>
<h2 id="build-shape"><a class="no-underline" href="#build-shape">Build shape</a></h2>
<p>The CMake project is <code>DotTalkpp</code> and uses C++20.</p>
<p>Common local configuration starts from these build options:</p>
<pre><code class="language-powershell">cmake -S D:\code\ccode -B D:\code\ccode\build -G "Visual Studio 17 2022" -A x64 -D DOTTALK_WITH_INDEX=ON -D DOTTALK_WITH_RELATIONS=ON -D BUILD_TESTING=ON
cmake --build D:\code\ccode\build --config Release --target dottalkpp
</code></pre>
<p>GUI lanes are optional:</p>
<pre><code class="language-powershell">cmake -S D:\code\ccode -B D:\code\ccode\build-wx-check -G "Visual Studio 17 2022" -A x64 -D DOTTALK_WITH_GUI=ON -D DOTTALK_WITH_WX=ON -D DOTTALK_WITH_INDEX=ON
</code></pre>
