# Career Lessons

Career lessons record what was learned building and maintaining LabTalk,
DotTalk++, x64base, xBase recovery work, documentation systems, proof systems,
and public learning material.

These are not automatically student lessons. A career lesson may become student
material only after the audience, evidence, source-memory boundary, and teaching
path are reviewed.
