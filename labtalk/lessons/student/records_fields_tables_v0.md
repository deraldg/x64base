# Records, Fields, and Tables v0

Status: draft
Audience: student, self-learner, instructor
Registry ID: lesson.student.records_fields_tables

## Purpose

Teach the beginner database vocabulary behind LabTalk and DotTalk++:

- table
- record
- field
- field type
- current work area
- current record state


## Student Path

1. Open the database literacy starter lab.
2. Identify the table being used.
3. List the fields and field types.
4. Read a small set of records.
5. Explain the difference between the table structure and the current record.

## Expected Observations

- A table has a structure before any one record is inspected.
- A record is one row inside the table.
- A field is one named piece of data inside each record.
- Runtime output should be treated as evidence, not just as a screen display.
